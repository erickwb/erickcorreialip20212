## Questão 01


R) Usando a sentença a + b + c ,podemo encontra duas derivações diferente (ou seja para a mesmam sentença temos duas arvores de análise)

Primeira forma: (derivação a esquerda)

 < S > -> < A >
        
       -> < A > + < A >
       
       -> < A > + < A > + < A >
       
       -> < id > + < A > + < A >
       
       -> a + < A > + < A >
       
       -> a + < id > + < A >
       
       -> a + b + < A >
       
       -> a + b + < id >
       
       -> a + b + c
       
Segunda forma: (derivação a esquerda)

 < S > -> < A >
        
       -> < A > + < A >
       
       -> < id > + < A >
       
       -> a + < A >
       
       -> a + < A > +  < A >
       
       -> a + < id > + < A >
       
       -> a + b + < A >
       
       -> a + b + < id >
       
       -> a + b + c
